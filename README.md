# Darren Knuckles — Portfolio Website

**Frontend Developer & Web Designer** | Chattanooga, TN

> Clean, Fast Websites That Help Businesses Stand Out

## About

Personal portfolio site for Darren Knuckles showcasing web design and development work for local small businesses.

## Features

- **Hero section** — name, title, and short value-prop message
- **Portfolio projects** — sample business websites with GitHub links
- **Services section** — business websites, landing pages, and redesigns
- **Contact form** — easy inquiry form for prospective clients
- **GitHub link** — links to [github.com/dlkilfaba-cmyk](https://github.com/dlkilfaba-cmyk)

## Tech Stack

Plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools. Fast by default.

## Running Locally

Open `index.html` in any browser. No build step required.
